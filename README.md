asciisciit
===========

ASCII Art, Video, and Plotting Toolbox

## Installation

### Requirements

You will need these three packages:

[Pillow](https://pillow.readthedocs.org/en/latest/)

[Numpy](http://www.numpy.org/)

[OpenCV](http://opencv.org/)

### Setup

Install with:

    python setup.py install
